String Handling in PHP

1. Interpolation with Curly Braces

String interpolation allows you to embed variables directly within string literals. In PHP, you can use curly braces to clarify variable boundaries within double-quoted strings.

Example:


    $name = "Alice";
    echo "Hello, {$name}!"; // Outputs: Hello, Alice!

2. Characters and String Indexes

Strings in PHP can be accessed using indexes, where the first character is at index 0. You can retrieve individual characters from a string using square brackets.

Example:


    $string = "Hello";
    echo $string[0]; // Outputs: H

3. String Operators

PHP provides several operators to manipulate strings:

  • Concatenation Operator: The dot (.) operator is used to concatenate two strings.
    • 
    $greeting = "Hello" . " World!"; // Outputs: Hello World!
  • Concatenation Assignment Operator: The .= operator appends a string to the existing string.
    • 
    $message = "Hello";
    $message .= " World!"; // Outputs: Hello World!

5. String Functions

PHP offers a variety of built-in string functions to perform common operations:

  • strlen(): Returns the length of a string.
    • 
    echo strlen("Hello"); // Outputs: 5
  • strtoupper(): Converts a string to uppercase.
    • 
    echo strtoupper("hello"); // Outputs: HELLO
  • strtolower(): Converts a string to lowercase.
    • 
    echo strtolower("HELLO"); // Outputs: hello

6. Formatting Strings

PHP provides functions for formatting strings, such as sprintf() and printf(), which allow you to create formatted output.

Example:


    $number = 5;
    $formattedString = sprintf("You have %d new messages.", $number);
    echo $formattedString; // Outputs: You have 5 new messages.

7. Comparing and Searching Strings and Substrings

PHP provides several functions to compare and search for substrings within strings:

  • strcmp(): Compares two strings.
    • 
    $result = strcmp("abc", "abc"); // Outputs: 0 (equal)
  • strpos(): Finds the position of the first occurrence of a substring.
    • 
    $position = strpos("Hello World", "W"); // Outputs: 6
  • substr(): Returns a portion of a string.
    • 
    $sub = substr("Hello World", 6, 5); // Outputs: World
