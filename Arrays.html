Arrays in PHP

1. PHP Arrays

An array is a data structure that allows you to store multiple values in a single variable. PHP supports both indexed arrays and associative arrays.

2. Creating Arrays

There are multiple ways to create arrays in PHP:

  • Using the array() function:
    • 
    $fruits = array("Apple", "Banana", "Cherry");
  • Using the shorthand syntax:
    • 
    $vegetables = ["Carrot", "Potato", "Onion"];

3. Accessing Array Elements

You can access individual elements in an array using their index. In PHP, indexes start from 0.

Example:


    $fruits = ["Apple", "Banana", "Cherry"];
    echo $fruits[1]; // Outputs: Banana

4. Multidimensional Arrays

A multidimensional array is an array that contains one or more arrays. It can be thought of as an array of arrays.

Example:


    $students = [
        ["name" => "Alice", "age" => 22],
        ["name" => "Bob", "age" => 20],
        ["name" => "Charlie", "age" => 23]
    ];
    echo $students[1]["name"]; // Outputs: Bob

5. Inspecting Arrays

To inspect arrays, you can use the print_r() and var_dump() functions, which provide detailed information about the array's structure and contents.

Example:


    print_r($fruits);
    var_dump($students);

6. Deleting from Arrays

You can remove elements from an array using the unset() function.

Example:


    unset($fruits[1]); // Removes "Banana"
    print_r($fruits); // Outputs: Array ( [0] => Apple [2] => Cherry )

7. Iterating with each() and foreach()

You can iterate through arrays using the foreach() loop, which is specifically designed for arrays. The each() function is deprecated in PHP 7.2. Use foreach() instead.

Example:


    foreach ($fruits as $fruit) {
        echo $fruit . " ";
    }
    // Outputs: Apple Cherry

8. Iterative Functions

PHP provides several functions for traversing arrays:

  • current(): Returns the current element of an array.
    • 
    echo current($fruits); // Outputs: Apple
  • next(): Moves the internal pointer to the next element and returns it.
    • 
    next($fruits);
    echo current($fruits); // Outputs: Cherry
  • prev(): Moves the internal pointer to the previous element and returns it.
    • 
    prev($fruits);
    echo current($fruits); // Outputs: Apple
  • reset(): Moves the internal pointer to the first element of an array.
    • 
    reset($fruits);
    echo current($fruits); // Outputs: Apple
  • end(): Moves the internal pointer to the last element of an array.
    • 
    end($fruits);
    echo current($fruits); // Outputs: Cherry
