Functions in PHP

1. Function Definition

A function is a block of code that performs a specific task and can be reused throughout the program. Functions help in organizing code, making it more readable and maintainable.

The basic syntax for defining a function is as follows:


    function functionName() {
        // Code to be executed
    }

2. Creating and Invoking User-Defined Functions

You can create user-defined functions to perform specific operations. After defining a function, you can invoke it by using its name followed by parentheses.

Example:


    function greet($name) {
        return "Hello, " . $name . "!";
    }
    
    echo greet("Alice"); // Outputs: Hello, Alice!

3. Formal Parameters versus Actual Parameters

Formal parameters are the variables listed as part of a function's definition, while actual parameters are the values you pass to the function when invoking it.

Example:


    function add($a, $b) { // $a and $b are formal parameters
        return $a + $b;
    }
    
    echo add(5, 10); // 5 and 10 are actual parameters

4. Function and Variable Scope

The scope of a variable determines where it can be accessed in the code. There are two main types of scope in PHP:

  • Local Scope: Variables defined within a function can only be accessed within that function.
    • 
    function test() {
        $localVar = "I am local";
        echo $localVar;
    }
    test(); // Outputs: I am local
    // echo $localVar; // Error: Undefined variable
  • Global Scope: Variables defined outside of any function can be accessed globally, but must be explicitly declared as global inside functions.
    • 
    $globalVar = "I am global";
    
    function showGlobal() {
        global $globalVar;
        echo $globalVar;
    }
    showGlobal(); // Outputs: I am global

5. Recursion

Recursion is a programming technique where a function calls itself to solve a problem. It is essential to have a base case to prevent infinite loops.

Example:


    function factorial($n) {
        if ($n <= 1) {
            return 1; // Base case
        } else {
            return $n * factorial($n - 1); // Recursive call
        }
    }
    
    echo factorial(5); // Outputs: 120

6. Library Functions

PHP comes with a rich set of built-in functions, known as library functions, that can perform various tasks. You can use these functions without needing to define them yourself.

Example:


    $length = strlen("Hello, World!"); // strlen() is a library function
    echo $length; // Outputs: 13
