PHP Basics

1. Tokens

In PHP, tokens are the smallest units of code that the parser recognizes. Tokens include keywords, variables, operators, and symbols. They form the building blocks of PHP scripts. Understanding tokens is essential for writing syntactically correct code.

2. Variables

Variables in PHP are used to store data. They are declared with a dollar sign ($) followed by the variable name. Variable names are case-sensitive and can contain letters, numbers, and underscores. For example:

$name = "John";

This creates a variable named $name that holds the string value "John".

3. Variable Scope

Variable scope determines the accessibility of variables in different parts of a program. PHP has four types of variable scope:

  • Local Scope: Variables declared inside a function are local to that function.
  • Global Scope: Variables declared outside of functions are in the global scope and can be accessed from anywhere in the script.
  • Static Scope: A static variable retains its value across function calls.
  • Parameter Scope: Variables passed as parameters to functions are accessible only within that function.

4. Constants

Constants are identifiers that hold fixed values. Unlike variables, constants cannot be changed once defined. They are declared using the define() function:

define("SITE_NAME", "My Website");

This creates a constant SITE_NAME with the value "My Website". Constants are often used for configuration values that should not change.

5. Data Types

PHP supports several data types, which can be categorized as:

  • String: A sequence of characters. Example: "Hello, World!"
  • Integer: A whole number. Example: 42
  • Float: A number with a decimal point. Example: 3.14
  • Boolean: A value that can be either true or false.
  • Array: A collection of values stored in a single variable.
  • Object: An instance of a class.
  • NULL: A special type that represents a variable with no value.

6. Number Handling in PHP

PHP provides several functions for handling numbers, including:

  • abs(): Returns the absolute value of a number.
  • round(): Rounds a floating-point number to the nearest integer.
  • rand(): Generates a random integer.
  • max(): Returns the highest value from a set of numbers.
  • min(): Returns the lowest value from a set of numbers.

7. Operands

Operands are the values on which operators perform actions. In PHP, operands can be variables, constants, or literals. For example:

$sum = $a + $b;

Here, $a and $b are operands.

8. Operators

PHP supports various operators, which can be classified into categories:

  • Arithmetic Operators: Used for mathematical operations (e.g., +, -, *, /, %).
  • Comparison Operators: Used to compare values (e.g., ==, !=, >, <, ===).
  • Logical Operators: Used to combine conditional statements (e.g., &&, ||, !).
  • Assignment Operators: Used to assign values (e.g., =, +=, -=).

9. Expressions

An expression is a combination of variables, constants, operators, and functions that PHP evaluates to produce a value. For example:

$result = ($a + $b) * $c;

This expression combines addition and multiplication to produce a result.

10. Operator Precedence

Operator precedence determines the order in which operations are performed in an expression. For example, multiplication and division have higher precedence than addition and subtraction. You can use parentheses to alter precedence:

$result = $a + $b * $c; // Multiplies first

To change the order, use parentheses:

$result = ($a + $b) * $c; // Adds first

11. Comments

Comments in PHP are used to annotate code, making it easier to understand. PHP supports single-line and multi-line comments:

  • Single-Line Comment: // This is a single-line comment
  • Multi-Line Comment: /* This is a multi-line comment */

12. Echo and Print Statements

The echo and print statements are used to output data to the screen:

  • echo: Can take multiple parameters and does not return a value. Example: echo "Hello, World!";
  • print: Can only take one argument and always returns 1. Example: print "Hello, World!";
