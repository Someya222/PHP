Forms in PHP

1. Working with HTML Form Controls and PHP

Forms are essential for collecting user input in web applications. HTML provides various form controls, such as text fields, checkboxes, radio buttons, and drop-down lists, which can be processed using PHP.

2. Superglobal Variables

Superglobal variables in PHP are built-in global arrays that allow you to access information about the request and user input. The most common superglobals include:

  • $_GET: Used to collect data sent in the URL query string.
  • $_POST: Used to collect data submitted via HTTP POST method.
  • $_REQUEST: Combines data from both $_GET and $_POST.
  • $_SESSION: Used to store session variables.
  • $_COOKIE: Used to access cookie variables.

3. Superglobal Array

Superglobal arrays allow you to access information without needing to declare them as global. For instance, you can access user input through the $_POST superglobal array as follows:

Example:


    // process.php
    $name = $_POST['name'];
    $email = $_POST['email'];
    echo "Name: $name, Email: $email";

4. Importing User Input

When a user submits a form, PHP can process the data using the superglobal arrays. It's essential to validate and sanitize user input to prevent security risks such as SQL injection or cross-site scripting (XSS).

Example of Importing and Sanitizing User Input:


    // process.php
    $name = htmlspecialchars($_POST['name']);
    $email = htmlspecialchars($_POST['email']);
    echo "Name: $name, Email: $email";

5. Accessing User Input

You can access user input through the superglobal arrays like this:

Example:


    if ($_SERVER["REQUEST_METHOD"] == "POST") {
        $name = $_POST['name'] ?? 'Guest';
        echo "Welcome, $name!";
    }

This code checks if the request method is POST and retrieves the name input, providing a default value if it's not set.
