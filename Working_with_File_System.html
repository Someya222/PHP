Working with File System in PHP

1. Understanding PHP File Permissions

File permissions in PHP determine who can read, write, or execute a file. Permissions are typically set using three attributes: owner, group, and others. They can be expressed in numeric (octal) format or symbolic format (e.g., 'r' for read, 'w' for write, 'x' for execute).

Example:


    // Set file permissions to read and write for owner, and read for others
    chmod('file.txt', 0644);

2. Opening and Closing a File

To work with files in PHP, you first need to open them using the fopen() function. You can specify the mode in which to open the file, such as read, write, or append. After completing the operations, it's important to close the file using the fclose() function.

Example:


    // Open a file for reading
    $file = fopen('file.txt', 'r');
    
    // Close the file
    fclose($file);

3. File Reading Functions

PHP provides several functions for reading files, such as:

  • fgets(): Reads a single line from the file.
  • fread(): Reads a specified number of bytes from the file.
  • file_get_contents(): Reads the entire file into a string.

Example:


    // Read the entire file into a string
    $content = file_get_contents('file.txt');
    echo $content;

4. File Writing Functions

To write data to a file, you can use:

  • fwrite(): Writes data to an open file.
  • file_put_contents(): Writes data to a file, creating it if it does not exist.

Example:


    // Open a file for writing
    $file = fopen('file.txt', 'w');
    
    // Write data to the file
    fwrite($file, "Hello, World!\n");
    
    // Close the file
    fclose($file);

5. File System and Directory Functions

PHP provides various functions to work with the file system and directories:

  • mkdir(): Creates a new directory.
  • rmdir(): Removes an empty directory.
  • scandir(): Returns an array of files and directories in a specified directory.
  • unlink(): Deletes a file.

Example:


    // Create a new directory
    mkdir('new_directory');
    
    // Scan a directory
    $files = scandir('new_directory');
    print_r($files);
    
    // Delete a file
    unlink('file.txt');
